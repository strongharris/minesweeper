# Minesweeper

## Installation

Pre-Req: latest node and yarn installed.

```
yarn install
yarn dev
open http://localhost:3000
```

## Features Included

Try the [demo](https://minesweeperjest-3s76qciij.now.sh):

* Mine sweeper game rules
* Timer
* Reset
* Change board size
* High scores (sorted by board size and time spent)
